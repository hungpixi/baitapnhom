# baitapnhom (Nhóm 2 - Nhập môn CNTT - UIT)

Dự án giới thiệu thành viên của **Nhóm 2** môn **Nhập môn Công nghệ Thông tin** tại **Trường Đại học Công nghệ Thông tin — UIT**.

## Thành viên nhóm
Dự án bao gồm thông tin giới thiệu của 15 thành viên trong nhóm.

## Công nghệ sử dụng
- **HTML / CSS / JavaScript** (Vanilla static site)
- Hiệu ứng Matrix Code Rain & Mô phỏng Terminal độc đáo.
- Nhạc nền và hiệu ứng âm thanh sống động (Keystrokes, Glitch, BSOD sound effects).

## Cách chạy dự án
Chỉ cần mở file `index.html` trực tiếp trên trình duyệt hoặc chạy một local server (ví dụ: Live Server trong VS Code).
